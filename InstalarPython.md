##Vamos a crear un environmet, lo pasas a serguir son:

1- Entrar en el terminal de Windows y descarger el paquete llamado 'virtualenv' con este comando: pip install virtualenv. Adjunto una foto de la resultado.

![imagen](https://github.com/Racriberny/IABD/assets/90463533/bfd02d08-075a-4d66-b1ef-d9e9fec39ec1)

2- Ahora cuando ya tenemos instalado el paquete tendremos que crear el environmet, poniendo en el termina: virtualenv (nombre)

3- Tendremos que activar el entonrno escribiendo: nombre_entorno\Scripst\Activate.

![imagen](https://github.com/Racriberny/IABD/assets/90463533/f7a866d9-25ba-48b2-9ace-b41ace6e7d9f)

4- Para desactivar el entorno y salir tienes que poner 'deactivate'

![imagen](https://github.com/Racriberny/IABD/assets/90463533/518f815d-f235-4776-9d0b-1c54286806ee)

5- Y con esto ya teneis como crear un environmet.
